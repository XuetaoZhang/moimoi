# Moimoi App 安装和演示指南

## 📱 第一步：安装到手机

### 方法1：使用USB连接安装（推荐）

#### 准备工作：
1. **在手机上开启开发者模式**
   - 打开手机「设置」→「关于手机」
   - 连续点击「版本号」7次
   - 看到提示"您已处于开发者模式"

2. **开启USB调试**
   - 返回「设置」→「系统」→「开发者选项」
   - 打开「USB调试」开关
   - 打开「USB安装」开关（部分手机需要）

3. **用USB线连接手机到电脑**
   - 手机会弹出「允许USB调试」提示
   - 点击「允许」或「确定」

#### 安装命令：
```bash
# 1. 检查手机是否连接成功
adb devices

# 应该看到类似输出：
# List of devices attached
# XXXXXX  device

# 2. 安装APK
adb install /Users/zhangxuetao/miaow/miaowapp/app/build/outputs/apk/debug/app-debug.apk

# 如果已安装旧版本，使用 -r 参数覆盖安装：
adb install -r /Users/zhangxuetao/miaow/miaowapp/app/build/outputs/apk/debug/app-debug.apk
```

---

### 方法2：直接复制APK文件到手机

#### 步骤：
1. **找到APK文件**
   ```
   /Users/zhangxuetao/miaow/miaowapp/app/build/outputs/apk/debug/app-debug.apk
   ```

2. **复制到手机**
   - 用USB连接手机（选择「文件传输」模式）
   - 或者通过微信/QQ/邮件发送到手机
   - 或者上传到网盘，手机下载

3. **在手机上安装**
   - 打开手机「文件管理」
   - 找到 `app-debug.apk` 文件
   - 点击安装
   - 如果提示「不允许安装未知来源应用」：
     - 点击「设置」
     - 允许该来源的应用安装

## 🎬 第二步：完整演示流程

### 演示前准备：
✅ 提前准备2-3张清晰的照片（人物或宠物）

### 完整演示步骤（3-5分钟）

#### 1️⃣ 启动应用（10秒）
```
打开手机 → 找到"moimoi"应用 → 点击打开
```
**展示点**：简洁的首页，空白形象卡片，"点击创建专属形象"提示

---

#### 2️⃣ 创建形象（30秒）
```
点击中央的虚线卡片 → 进入上传页面
点击相机图标 → 从相册选择照片
（建议选择清晰的正面照）
点击"下一步：AI伴侣生成"按钮
```
**展示点**：
- 清晰的上传界面
- 隐私保护提示
- 按钮只有选择照片后才能点击

---

#### 3️⃣ AI生成中（30-60秒）
```
自动进入加载页面
显示"AI正在唤醒你的专属伙伴..."
圆形进度条动画
显示两个风格选项：色彩斑斓、渐变写实
```
**展示点**：
- 优雅的加载动画
- 清晰的状态提示
- **这时可以介绍技术实现：OpenAI API、Q版形象生成**

**⚠️ 注意**：这个过程需要30-60秒，可以提前录制加速版本，或者在演示时跟观众互动介绍功能

---

#### 4️⃣ 主题画廊（30秒）

自动跳转到主题画廊页面

（站立、打招呼、睡觉等不同状态）
选择最喜欢的一张

**展示点**：
- 流畅的滑动效果
- AI生成的3张不同状态的Q版形象
- 主题标签和装饰元素

#### 5️⃣ 同步成功（10秒）

**说明**：这些是扩展功能，可以实现更深度的系统集成

## ⚠️ 常见问题处理

### 问题1：AI生成太慢
**方案**：
- 提前生成好形象，演示时直接展示
- 或者播放提前录制的加速视频
- 在等待时介绍技术实现

### 问题2：网络无法访问API
**方案**：
- 使用手机热点
- 或者演示提前录制的视频
- 代码中已有占位图，可以继续演示其他功能

### 问题3：手机权限问题
**方案**：
- 提前在演示手机上测试一遍
- 确保所有权限都已授予
- 壁纸功能一般都能正常工作
